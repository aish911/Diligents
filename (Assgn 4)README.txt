SVC (Simple Version Control)

1. Initialize the cnt_file.txt by 0.

2. Have some data into a.txt.

Initially ,there is a sample file with some data in it.

On running the program ,it allows you to append in or delete from the file.

If append is selected ,the file with greater version will get created and data will get appended in this file.

If delete is selected ,the string you will enter will get deleted.

For each update in the sample file ,a new version of that file will get created.
